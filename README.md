# Design and Development of Generative AI Applications

**Spring 2026 | UNC Charlotte**

## Course Overview

This course provides a comprehensive introduction to designing and developing generative AI applications. Students will bridge the gap between theoretical knowledge and practical application, learning fundamental concepts necessary to architect robust applications powered by foundational models.

### What You'll Learn

- **Advanced RAG Pipelines:** Optimize data retrieval to ensure models provide accurate, context-aware responses
- **Agentic Systems:** Design autonomous agents that can reason, plan, and execute multi-step tasks
- **Model Context Protocol (MCP):** Connect applications to external data sources and tools to expand model capabilities
- **Scalable Deployment:** Gain hands-on experience with containerization and cloud orchestration for production environments
